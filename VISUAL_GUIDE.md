Opacity**: /10, /20, /30, /50 para refinamento\n- **Shadow**: shadow-sm (leve) ao invés de pesada\n- **Border Opacity**: border-zinc-800/50 para suavidade\n\n### Micro-interactions\n- **Hover**: Sempre presente, suave, contextual\n- **Focus**: Ring sutil com cor primary\n- **Active**: Scale 95% para feedback\n- **Disabled**: Opacidade 50%, sem cursor\n\n### Typography\n- **Labels**: UPPERCASE, font-semibold, tracking-wide (premium feel)\n- **Titles**: font-bold, tracking-tight (control)\n- **Body**: font-normal, clear hierarchy\n- **Secondary**: text-zinc-500/600 (discreta)\n\n---\n\n## 🎯 Checklist Visual\n\nAo adicionar nova feature, verificar:\n\n- [ ] Labels em uppercase com tracking-wide\n- [ ] Inputs rounded-lg com border-zinc-700/50\n- [ ] Focus states com ring-primary/30\n- [ ] Hover states em tudo interativo\n- [ ] Transições 200ms smooth\n- [ ] Spacing: gap-4 ou space-y-4\n- [ ] Padding: p-5 sm:p-6\n- [ ] Borders: border-zinc-800/50\n- [ ] Text colors: zinc-{300,400,500}\n- [ ] Responsive: sm:, md:, lg:\n- [ ] Icons: h-4 w-4 ou h-5 w-5 (not bigger!)\n- [ ] Badges: com borders e backgrounds suaves\n\n---\n\n## 📚 Referência Rápida\n\n### Cores
